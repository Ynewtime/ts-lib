## Usage

Install package:

```sh
# npm
npm install @ynewtime/ts-lib

# yarn
yarn add @ynewtime/ts-lib

# pnpm
pnpm install @ynewtime/ts-lib
```

Import:

```js
// ESM
import { sayHello } from '@ynewtime/ts-lib'

// CommonJS
const { sayHello } = require('@ynewtime/ts-lib')
```
